## 项目经验

### 1. AI坊学生管理系统（全栈开发）
**2025.09 - 至今** | 厦门工学院人工智能创作坊

**项目背景**：为人工智能创作坊开发一套学生管理系统，实现学生信息管理、分时段签到、数据可视化等功能。

**技术栈**：
- **前端**：Vue3、Element Plus、ECharts、Pinia、Vue Router
- **后端**：Spring Boot 3.x、Spring Security + JWT、Redis、MySQL、MinIO
- **部署**：Docker、Nginx

**核心功能**：
- 学生信息管理模块（CRUD操作、权限控制）
- 分时段拍照签到系统（早/午/晚三个时段，内网限制）
- 大屏数据可视化（打卡排名、年级/专业分布）
- 验证码管理系统（自动刷新、状态管理、权限检查）

**关键成就**：
- 通过IP白名单+JWT双重校验，确保系统安全性
- 基于Redis缓存，将重复签到判断响应时间优化至100ms内
- 实现多端适配，支持手机端和电脑端访问
