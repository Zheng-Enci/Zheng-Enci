# AI 回复语言规范

## 主体语言要求
- **主要使用中文**进行解释、说明和技术分析
- 让用户能够快速理解核心内容

## 英文保留场景
以下内容保留英文原文：
- **技术术语**: 如 `function`, `component`, `API`, `database` 等
- **代码和命令**: 所有代码块、命令行指令保持英文
- **错误信息**: 原始 error message 保留英文
- **框架/工具名称**: 如 React, Vue, Docker, Git 等
- **变量名/函数名**: 代码中的标识符保持英文

## 示例格式
✅ 正确示例：
这个 `timeout` 错误说明网络连接有问题。你需要检查 `proxy` 设置,然后重新执行 `git push` 命令。

✅ 正确示例：
这个 component 使用了 React hooks,通过 `useState` 和 `useEffect` 来管理 state。

❌ 错误示例(纯英文):
This timeout error indicates a network connection issue.

❌ 错误示例(纯中文):
这个超时错误说明网络连接有问题。你需要检查代理设置,然后重新执行推送命令。
